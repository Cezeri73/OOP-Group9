Welcome to our Object-\/\+Oriented Programming (OOP) project repository, developed as part of the CENG201 course at Ankara Yıldırım Beyazıt University. This project demonstrates key OOP concepts using C++.\hypertarget{md__r_e_a_d_m_e_autotoc_md1}{}\doxysection{\texorpdfstring{🚀 Project Overview}{🚀 Project Overview}}\label{md__r_e_a_d_m_e_autotoc_md1}
In this project, we designed and implemented a {\bfseries{Smart Home Automation System}}, which applies fundamental OOP principles to manage smart home devices such as lights, thermostats, and security cameras. The system allows users to monitor and control these devices remotely.\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysubsection{\texorpdfstring{Key Features}{Key Features}}\label{md__r_e_a_d_m_e_autotoc_md2}

\begin{DoxyItemize}
\item {\bfseries{Device Control\+:}} Users can turn devices on/off and set timers.
\item {\bfseries{Monitoring\+:}} Devices send alerts, such as motion detection alerts from security cameras.
\item {\bfseries{Observer Pattern\+:}} Devices send notifications when they detect changes (e.\+g., motion detected).
\item {\bfseries{Factory Pattern\+:}} Used to instantiate different types of smart devices dynamically.
\item {\bfseries{MVC Pattern\+:}} Separates device logic (Model), control logic (Controller), and the user interface (View) to enhance modularity and maintainability.
